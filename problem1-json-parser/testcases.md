# テストケース：JSONパーサ（問題1）

各言語の実装に対して、以下を実行して判定する。
「入力」を渡したとき「期待」の通りになれば合格。

## 正常系（OK＝正しい値を返す）

| # | 入力 | 期待 |
| --- | --- | --- |
| 1 | `null` | null |
| 2 | `true` | 真 |
| 3 | `false` | 偽 |
| 4 | `42` | 整数 42 |
| 5 | `3.14` | 小数 3.14 |
| 6 | `"hello"` | 文字列 hello |
| 7 | `"a\"b"` | 文字列 a"b |
| 8 | `[]` | 空配列 |
| 9 | `{}` | 空オブジェクト |
| 10 | `[1, 2, 3]` | 要素3つの配列 |
| 11 | `{"a": 1, "b": true}` | キー2つのオブジェクト |
| 12 | `{"a": [1, {"b": null}]}` | ネスト構造が正しく復元される |
| 13 | ` 42 ` (前後に空白) | 整数 42 |

## 異常系（ERR＝エラーになる。異常終了はNG）

| # | 入力 | 期待 |
| --- | --- | --- |
| 14 | (空文字列) | エラー |
| 15 | `[1, 2, ]` | エラー |
| 16 | `[1, 2` | エラー |
| 17 | `{"a": 1` | エラー |
| 18 | `01` | エラー |
| 19 | `1.` | エラー |
| 20 | `.5` | エラー |
| 21 | `foo` | エラー |
| 22 | `"abc` | エラー |

合計 22 ケース（正常系13・異常系9）。
`final_pass_count` は「通った数 / 22」で記録する。
