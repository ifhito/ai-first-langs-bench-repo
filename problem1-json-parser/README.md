# 問題1：JSONパーサ

文字列をJSONとして解釈する関数を、各言語でAIに実装させて比較します。
境界条件（末尾カンマ、閉じ括弧不足、不正な数値など）をどれだけ正しく処理できるかを測ります。

## このフォルダの中身

| ファイル | 役割 |
| --- | --- |
| `AGENT_INSTRUCTIONS.md` | エージェントへの実行指示（メイン） |
| `spec.md` | 課題の仕様（全言語共通） |
| `testcases.md` | 22個のテストケース（正常系13・異常系9） |
| `prompts/` | 言語ごとのプロンプト雛形 |
| `results/` | 実行結果がここに溜まる（エージェントが生成） |
| `REPORT.md` | 最終レポート（エージェントが出力） |

## 実行方法

このフォルダをAIコーディングエージェントに渡し、こう指示します。

> `AGENT_INSTRUCTIONS.md` を読んで、その通りにベンチマークを最後まで自動実行し、`REPORT.md` を出力してください。

詳しくはリポジトリ直下の [README](../README.md) を参照してください。
